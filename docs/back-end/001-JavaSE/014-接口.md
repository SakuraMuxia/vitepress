# 接口

接口：类似与上电脑主机上接口，用于连接对接其他子系统，相当于一个桥梁，用于连接两个系统。这个桥梁有一种约定，一种规则，一种能力。

```java
完成一件事情：要求+实现
只关注约定，不关注具体实现。
```

## 接口的规则

> 接口中的方法默认都为全局抽象方法
>
> 接口中的属性默认都是全局静态常量 public static final int a
>
> 接口不能直接new对象，
